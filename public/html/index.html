Flower auf Dauer - Backend

About

Flower auf Dauer ist ein Projekt der 5AAIF/6AAIF welches wie eine Art Management und Überwachungssystem für Pflanzen wirkt. Durch konstante Messungen der Umgebung von Pflanzen

Diese Seite dient als Gedätchnis von Flower auf Dauer. Hiermit werden alle Daten abgefragt, geupdated, gespeichert oder gelöscht (CRUD)

Verwendung

Über verschiedene Routen lassen sich Daten abfragen, suchen, erstellen löschen und updaten

Zeigt alle Users im Json-Format an

/users

Gibt den User mit übereinstimmender id im Json-Format aus

/users/id

Gibt einen oder mehrere User mit dem übereinstimmenden suchstring im Json-Format aus

/users/suchstring

Ladet alle User im Csv-Format herunter

/users/download

Ladet user mit übereinstimmender id im Csv-Fromat herunter

/users/id/download

Ladet user mit übereinstimmendem suchstring im Csv-Format herunter

/users/suchstring/download

Zeigt alle Fields im json-Format an

/fields

Zeigt das Field mit übereinstimmender id im Json-Format an

/fields/id

Gibt ein oder mehrere Fields mit übereinstimmendem suchstring im Json-Format zurück

/fields/suchstring

Ladet alle Fields im Csv-Format herunter

/fields/download

Ladet das Field mit übereinstimmender id im Csv-Format herunter

/fields/id/download

Ladet die fields mit übereinstimmendem suchstring im Csv-Format herunter

/fields/suchstring/download

Zeigt alle Sensoren im Json-Format an

/sensors

Zeigt Sensor mit übereinstimmender id im Json-Format an

/sensors/id

Zeigt alle Sensoren mit übereinstimmendem suchstring im Json-Format an

/sensors/suchstring

Ladet alle Sensoren im Csv-Format herunter

/sensors/download

Ladet den Sensor mit übereinstimmender id im Csv-Format herunter

/sensors/id/download

Ladet alle Sensoren mit übereinstimmendem suchstring im Csv-Format herunter

/sensors/temp/download

Zeigt alle Sensorwerte im Json-Format an

/sensorvalues

Zeigt Sensorwert mit übereinstimmender id im Json-Format an

/sensorvalues/id

Lädt alle Sensorwerte im Csv-Format herunter

/sensorvalues/download

Lädt den Sensorwert mit übereinstimmender id im Csv-Format herunter

/sensorvalues/id/download
